# WorkSphere – Gestion du Personnel #
## Description
WorkSphere est une application web pour gérer le personnel sur un plan d’étage.  
Elle permet d’ajouter, déplacer et supprimer des employés, en respectant les rôles et les zones autorisées.
## Fonctionnalités
- Ajouter et supprimer des employés
- Affectation automatique selon le rôle et la zone
- Visualisation des employés non assignés
- Profil détaillé des employés (photo, email, téléphone, expériences)
- Interface responsive pour ordinateur, tablette et mobile
- Limitation du nombre d’employés par zone
- Zones vides visibles pour mieux organiser l’espace
## Technologies
- HTML, CSS, JavaScript
- Flexbox pour le design
- Animations CSS pour fluidité
- Déploiement possible sur GitHub Pages
